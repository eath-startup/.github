# Organization GitHub Configuration

This repository contains the global configuration and templates used across all repositories in this GitHub organization.

Its purpose is to standardize how the team reports issues, proposes features, and submits pull requests.

---

## Purpose

This repository provides:

- Global **Issue Templates**
- A **Pull Request Template**
- Contribution guidelines used across the organization

By centralizing these templates, we ensure that all projects follow the same workflow and maintain consistent documentation.
